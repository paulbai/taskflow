# Collaborative To-Do List

## Setup Instructions

Since the automatic installation was blocked, please run the following commands in your terminal to get started:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   npm start
   ```

The app will be available at [http://localhost:3000](http://localhost:3000).
